Exception异常

异常页用于对页面特定的异常状态进行反馈。通常，它包含对错误状态的阐述，并向用户提供建议或操作，避免用户感到迷失和困惑。

引用方式：

import Exception from 'ant-design-pro/lib/Exception';

详细使用方式请参照：独立使用 pro 组件

代码演示

404

抱歉，你访问的页面不存在
404

404 页面。

import Exception from 'ant-design-pro/lib/Exception';

ReactDOM.render(
  <Exception type="404" />
, mountNode);

500

抱歉，服务器出错了
500

500 页面。

import Exception from 'ant-design-pro/lib/Exception';

ReactDOM.render(
  <Exception type="500" />
, mountNode);

403

抱歉，你无权访问该页面
403

403 页面，配合自定义操作。

import Exception from 'ant-design-pro/lib/Exception';
import { Button } from 'antd';

const actions = (
  <div>
    <Button type="primary">Home</Button>
    <Button>Detail</Button>
  </div>
);
ReactDOM.render(
  <Exception type="403" actions={actions} />
, mountNode);
